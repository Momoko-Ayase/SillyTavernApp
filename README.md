# SillyTavernApp

Run SillyTavern on your Android device, without the need to execute commands in Termux.

## Building

See [BUILDING.md](./BUILDING.md).

## Data location

All user data lives at `/storage/emulated/0/Documents/SillyTavern` (`characters/`, `chats/`,
`settings.json`, …) and survives app restarts and reinstalls.
